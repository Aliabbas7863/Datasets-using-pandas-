# Programming Language Popularity Ratings Dataset

## Overview

This repository contains a dataset that explores the popularity of major programming languages using multiple indicators such as developer usage, community activity, and job demand. The dataset also includes the year each programming language was introduced, providing additional context for analyzing technology trends over time.

The goal of this dataset is to provide a structured resource that can be used for data analysis, visualization, and educational projects related to programming language adoption and popularity.

## Dataset Description

Each row in the dataset represents a programming language along with several indicators used to estimate its overall popularity. The dataset combines different metrics to calculate a popularity rating and ranking for each language.

The dataset includes information about both modern and legacy programming languages, allowing users to explore how programming technologies have evolved.

## Dataset Features

The dataset contains the following columns:

- **Language** – Name of the programming language  
- **Year Introduced** – The year the language was first released  
- **StackOverflow Popularity** – Indicator representing developer discussions and community engagement  
- **Job Demand Index** – Estimated demand for the language in job listings  
- **Developer Usage Score** – Estimated level of developer adoption  
- **Popularity Rating** – Overall rating calculated from the combined indicators  
- **Rank** – Ranking of languages based on their popularity rating
